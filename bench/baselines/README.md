# Benchmark Baselines

This directory will contain baseline comparison data once the FSM harness is exercised against a real SWE-bench Verified instance set.

v1.2.2 ships the harness scaffold only. Future versions will:

- Run the harness against a set of SWE-bench instances
- Capture per-instance results in `bench_result.json` files
- Aggregate results into `run_<timestamp>.json` summary files
- Store baseline snapshots here for comparison across releases

No baseline data is currently available.
